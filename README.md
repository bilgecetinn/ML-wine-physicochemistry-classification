# ML-wine-classification
ANN-based binary classifier distinguishing red and white Portuguese Vinho Verde wines from physicochemical properties. Features feature selection, k-fold cross-validation, and SHAP explainability analysis. Graded assignment for Prof. Romuald Poteau's AI &amp; ML in Chemistry module, LTTC Winter School Nov 2024 (TCCM Master's), Aspet, Toulouse, France.
